# UPnP Client for Dart
Original package: https://github.com/SpinlockLabs/upnp.dart
Null safety added by: https://github.com/thecoda/upnp.dart

- Discovery
- Device Information
- Services
- Actions
- Host Support
- DIAL API
- Router API
